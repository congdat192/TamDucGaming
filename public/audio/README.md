# Audio Assets

This directory contains all audio files for the Santa Jump game.

## BGM (Background Music)
- `homepage.mp3` - Homepage background music
- `gameplay.mp3` - Game background music  
- `leaderboard.mp3` - Leaderboard background music

## SFX (Sound Effects)
- `jump.mp3` - Jump sound
- `collect-gift.mp3` - Collect gift sound (+10)
- `collect-glasses.mp3` - Collect glasses sound (+20)
- `collect-star.mp3` - Collect star sound (+30)
- `hit-bomb.mp3` - Hit bomb sound (-30)
- `game-over.mp3` - Game over sound
- `button-click.mp3` - Button click sound
- `modal-open.mp3` - Modal open sound
- `achievement.mp3` - Achievement sound

## Audio Sources (Royalty-Free)
- Freesound.org
- Pixabay Music
- YouTube Audio Library
- Incompetech (Kevin MacLeod)
- Bensound

## Format
- Primary: MP3 (best browser compatibility)
- Sample Rate: 44.1kHz
- Bitrate: 128kbps (BGM), 96kbps (SFX)
